Este archivo proporciona directrices para el diseño de la interfaz de usuario.

-- Los componentes visuales se deben diseñar de forma compacta como las aplicaciones actuales que organizan el contenido de forma organizada, agradable a la vista, minimalista y facil de comprender al usaurio sin abrumarlo 
-- Usar los colores definidos en el tema de la aplicación (ColorScheme) para mantener la coherencia visual en toda la interfaz de usuario.
-- Usar las fuentes y tamaños de texto definidos en el tema de la aplicación (TextTheme) para asegurar la legibilidad y coherencia en toda la interfaz de usuario.
